String类

其他类型到String的转换

  • 例如： int i-123; 转换为String类型，有以下方式
    • String s = i+""
    • String s = String.valueof(i);

其他类型到String

StringBuilder

  • String需要频繁操作时，效率十分低下， 这时候应该采用 StringBuilder。

​ StringBuilder是 一个可变的字符序列。此类提供一个与 StringBuffer 兼容的 API，但不保证同步

继承方式与访问权限

  1. java中都是单继承的，即：每个子类只有一个直接父类， 但是可以多重继承，即垂直继承。单继承可以避免出现菱形继承的问题， 避免数据出现混乱。
